# Conway Logistics CRM

A full-stack Logistics Customer Relationship Management (CRM) platform built for managing transportation enquiries, admin operations, notifications, audit logs, and login activity.

## Features

* JWT Authentication & Authorization
* Role-Based Access Control
* Dashboard Analytics
* Enquiry Management
* Enquiry Notes
* Notifications System
* Admin Management
* Audit Logs
* Login History Tracking
* Material UI Design
* React Query Data Fetching
* Responsive UI

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Material UI
* React Router
* React Query
* Axios
* React Toastify

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* JPA / Hibernate
* MySQL

## Roles

### Super Admin

* Manage Admins
* View Audit Logs
* View Login History
* Full System Access

### Admin

* Manage Enquiries
* Add Notes
* View Notifications
